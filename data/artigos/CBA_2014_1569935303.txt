Abstract This paper proposes a new methodology to assist system planners, which is able to provide a set of good solutions to the
addition of new transmission lines, in order to reinforce the sub-transmission network. To circumvent the combinatorial explosion of
reinforcement alternatives, a constructive heuristic algorithm is used for adding new branches to the sub-transmission grid. Based on
performance indices to measure the attractiveness of existing and new branches and using an expansion tree, the proposed algorithm is
able to capture the combined effect of reinforcement additions. For selecting the best alternative, from the set of good reinforcement
solutions, the planned network performance in meeting the future load demand is evaluated, considering the following operational aspects ohmic losses, reliability, branch loading condition, and voltage profile. The evaluation of the performance indices and all operational aspects is achieved by applying a non-linear AC model-based power flow. For the reliability assessment, an enumeration technique is used to select system states. A large sub-transmission Brazilian system belonging to CEMIG utility is used to validate the
methodology and the results are presented and extensively discussed.
Keywords Sub-transmission planning, Transmission losses, Constructive algorithm, Reliability.
Resumo Este artigo propõe uma metodologia de auxílio aos planejadores de sistemas_de_subtransmissão, adaptada s suas peculiaridades e capaz de indicar boas alternativas de expansão. Para contornar a explosão combinatória de alternativas, um algoritmo heurístico construtivo é utilizado para adicionar reforços  rede de subtransmissão. Baseado em índices de desempenho, que avaliam a atratividade de ramos existentes e novos, e utilizando uma árvore de expansão, o algoritmo proposto é capaz de capturar o efeito combinado da adição de reforços. Para selecionar as melhores alternativas, de um conjunto de boas soluções de reforços, o desempenho da
rede planejada no atendimento da carga futura é avaliado sob os seguintes aspectos operativos perdas_ôhmicas, confiabilidade, perfil
de tensão nas barras e nível de carregamento dos circuitos. A avaliação de todos estes aspectos, assim como a obtenção dos índices de
desempenho é realizada por meio da aplicação do modelo não_linear (AC) de fluxo_de_potência. Para a avaliação_da_confiabilidade,
uma técnica de enumeração é utilizada para selecionar os estados do sistema. Um sistema real de subtransmissão da CEMIG é utilizado para teste e validação da metodologia proposta. Todos os resultados obtidos são amplamente discutidos.

identificar boas alternativas de expansão para os
sistemas_de_subtransmissão. Em geral, estes sistemas
utilizam níveis de tensão menores em comparação

perdas_ativas produz sistemas_de_subtransmissão
mais robustos e com maior capacidade para absorver

2.3 Algoritmo Proposto
Nesta subseção é descrito o Algoritmo Heurístico Construtivo (AHC), baseado em uma árvore de
busca e proposto para a obtenção das melhores alternativas de expansão. A maioria dos aspectos considerados neste trabalho é avaliada pela manipulação
de resultados obtidos via estudo de fluxo_de_potência.
Para esta finalidade foi utilizado o programa de Fluxo de Potência ANAREDE (CEPEL, 2011), disponibilizado para a Cemig. Cabe destacar que a ideia
central deste trabalho é propor uma metodologia
baseada nas ferramentas rotineiramente utilizadas
pelas empresas do setor_elétrico, aumentando as possibilidades de sua absorção pelos planejadores.
O algoritmo proposto é capaz de capturar a atratividade dos ramos (em termos dos respectivos custos de investimento e da redução_das_perdas_ativas)
para a Configuração Base do sistema e em situações
posteriores, ou seja, após a adição de reforços  malha do sistema. A ideia é ir selecionando Ramos
Candidatos, tanto para a Configuração Base, quanto
para as possíveis configurações reforçadas, i.e., após
a adição de 1 reforço, 2 reforços e assim por diante.
Na fase inicial do algoritmo são realizadas ações
visando eliminar a presença de configurações reforçadas semelhantes, situação altamente prejudicial ao
desempenho da metodologia, pois reduz a amplitude
da busca em árvore. Para tanto, é necessário identificar a presença de ramos novos semelhantes (i.e., que
tenham uma mesma barra terminal, sendo as segundas barras terminais interligadas e próximas, geralmente localizadas em um mesmo município). Adicionalmente, deve-se escolher dentre os ramos semelhantes identificados, aquele a ser mantido. O critério adotado neste trabalho é selecionar o ramo com a
maior soma dos índices Ind e IndV normalizados
(i.e., em por unidade, ou pu, do maior valor obtido
para cada índice). Destaca-se que, ao final do processo de busca, para cada uma das melhores configurações obtidas, é realizada a recuperação dos respectivos ramos semelhantes descartados. Deste modo, toda e qualquer configuração semelhante a uma
das melhores soluções encontradas será avaliada e a
substituirá, caso apresente um melhor desempenho.

4 Conclusão
Neste artigo é proposta uma nova metodologia
para o planejamento de sistemas_de_subtransmissão,
a qual utiliza um algoritmo heurístico construtivo
baseado numa busca em árvore. Para identificar os
melhores reforços, dois novos índices de desempenho são utilizados. Vale ressaltar que a estratégia de
planejamento adotado baseia-se na rede_de_transmissão principal projetada para o ano horizonte, o que é
mais adequado para sistemas de países com grande
território, como o Brasil. Outro ponto relevante é a
modelagem da rede AC, que é essencial para a análise tanto de fluxos de potência_ativa e reativa quanto
de aspectos operativos importantes para o planejamento de médio_prazo, ou seja, entre 5 e 10 anos.
Os resultados obtidos confirmam a eficácia dos
índices propostos para identificar os reforços mais
atrativos, a fim de reduzir as perdas_ativas e minimizar os custos de investimento. Também é possível
garantir que as configurações com perdas reduzidas
têm bons desempenhos em outros aspectos operativos, tais como confiabilidade, perfil_de_tensão e nível
de carregamento. O alto_desempenho computacional
obtido torna a metodologia adequada para lidar com
redes de subtransmissão de grande porte.
